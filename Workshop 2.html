Data analysis¶

The algorithms used in the analysis of all datasets are:

  • K-Nearest Neighbors (KNN)
  • Naive Bayes (Gaussian)
  • Decision tree (DTC)
  • Multilayer Perceptron (MLP)

First we divide the datasets in Y (target column) and X (features columns). X and Y are list and each position of the list stores the data corresponding to one dataset.

In [ ]:
# All columns except target
X = [None] * 10
# Target column
Y = [None] * 10

# Fill X and Y for each dataset
for i, dataset in enumerate(datasets):
    X[i] = dataset.loc[:, dataset.columns != 'target'] 
    Y[i] = np.array(dataset.loc[:, dataset.columns == 'target']).ravel()

Here we analyze whether the classes of each dataset are balanced or not and determines the evaluation measure best suited to the datasets.

In [ ]:
import matplotlib.pyplot as plt

# Type of evaluation
scoring_choosed = [None] * 10

for i, dataset in enumerate(datasets):
    # To plot the proportion of classes, we first count the values of the 'target' column and transform them into a frame
    class_distribuition = pandas.DataFrame(dataset['target'].value_counts())
    
    # Plot classes
    plt.figure()

    # We use the frame created above to plot the frequency of each class in the dataset
    plot = seaborn.barplot(x=class_distribuition.index, y='target', data = class_distribuition)
    plot.set_title(datasets_names[i])

    # Verify distribution of target classes
    total_sum = sum(class_distribuition['target'])
    partition = total_sum/len(class_distribuition['target'])
    for value in class_distribuition['target']:
        if(0.9 * partition <= value <= 1.1 * partition):
            scoring_choosed[i] = "accuracy" 
        else:
            scoring_choosed[i] = "balanced_accuracy"
            break

from math import sqrt

absolute_difference = [None] * 10

for i, algs in enumerate(datasets_algs):
    mean_err_A0 = 1 - algs[0]["results"]["mean_test_score"][0]
    std_A0 = algs[0]["results"]["std_test_score"][0]

    mean_err_A1 = 1 - algs[1]["results"]["mean_test_score"][0]
    std_A1 = algs[1]["results"]["std_test_score"][0]

    dif_mean_err = mean_err_A0 - mean_err_A1
    std = sqrt( (std_A0**2 + std_A1**2) / 2.0 )
    absolute_difference[i] = dif_mean_err / std
